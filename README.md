# ProjectRun

An educational mobile game that helps young children (ages 2-6) understand what happens during a nuclear medicine examination at Chelsea & Westminster Hospital. By guiding an animated panda character through each step of the visit, the game reduces anxiety and makes the hospital experience less intimidating.

Built as a group project at King's College London in collaboration with Chelsea & Westminster Hospital's Nuclear Medicine Department.

## Game Flow

1. **Introduction** — A letter arrives inviting the child to the hospital
2. **Anaesthetic Cream** — Apply numbing cream to the panda's arms and legs
3. **Waiting Room** — Explore the waiting area
4. **Food Game** — Learn about eating before the procedure
5. **Injection** — See how the injection works (with the radioactive tracer)
6. **Clock Scenes** — Wait for the tracer to work (45 min / 3 hr depending on procedure)
7. **Scanning Room** — Experience the scanning bed and camera
8. **Toilet Scene** — Post-scan bathroom visit
9. **Celebration** — Game complete!

The fully implemented path follows the **DMSA** scan protocol. The Meckel and Renogram branches have placeholder routing.

## Features

- Interactive mini-games at each stage of the hospital visit
- Animated panda character that walks, reacts, and guides the child
- Multiple procedure paths (DMSA fully implemented)
- Sound effects and background music throughout
- Designed for touch input (mobile-friendly)
